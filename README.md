# 🎯 Sentiment Analysis Tool

## 📌 Overview
This project is a Machine Learning-based Sentiment Analysis Tool that classifies text into Positive, Negative, or Neutral sentiments using Natural Language Processing (NLP).

The model is trained using Logistic Regression and processes text data using basic preprocessing techniques.

## 🚀 Features
- Text preprocessing (lowercasing, punctuation removal, stopword removal)
- Feature extraction using CountVectorizer
- Sentiment classification using Logistic Regression
- Accepts real-time user input
- Displays prediction results instantly
- Shows model accuracy

## 🛠️ Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn

## ⚙️ How It Works
1. Dataset is loaded from data.csv  
2. Text is cleaned and preprocessed  
3. Converted into numerical form using CountVectorizer  
4. Model is trained using Logistic Regression  
5. User inputs text  
6. Model predicts sentiment  

## ▶️ How to Run
1. Install libraries:

2. Run:
python project2.py

3. Enter any sentence to get prediction
